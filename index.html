    1. Introduction to greatname.net

      greatname.net is a simple project website for my personal projects. I could make something bigger but I prefer to keep it small since I am a student and have limited resources and time. :P

    2. The story behind the name

      The title of this chapter might make you believe that there is a deep and complicated story behind the name, but in reality, there isn't. It's just a name I liked and stuck with it. It's not a very creative name, but I like it and also, it's easy to remember.

    3. My current setup

      I use a pretty basic setup, at least that's what I think. My main machine is a desktop computer running Gentoo Linux. It's probably worth mentioning that I do use a laptop too for travel. For that I have an old Macbook Pro running Debian. I write most of my code in Visual Studio Code but I do sometimes use Vim and nano. In case anyone is wondering, I do not enjoy ricing my systems as it is not for me but I appreciate the work some people put into it.

    4. Why I do what I do

      I got into programming and web development because I enjoy creating and building things. I know this might sound cliché, but it's true. There isn't really a deeper reason.

      Honestly, I'm not sure. LOL! I kinda just go with the flow and see where it takes me. A lot of the time greatname.net just sits dormant since I genuinely forget that it exists.
